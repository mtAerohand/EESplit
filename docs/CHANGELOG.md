# 変更履歴

LiveSplitOneからの主な変更点を記録します。

## [Unreleased]

### 初期セットアップ
- LiveSplitOneをベースにEESplitプロジェクトを作成
- 独自リポジトリ (https://github.com/mtAerohand/EESplit.git) への移行完了

### 計画中の機能（要件定義完了: 2026-02-14）

#### 精鋭狩りRTA専用機能
1. **区間ごとの精鋭数設定機能**
   - スプリット編集UIに精鋭数入力フィールドを追加
   - 全区間の精鋭数合計表示

2. **平均討伐秒数の算出・表示**
   - 各区間で「秒/体」を計算（区間タイム ÷ 精鋭数）
   - スプリット一覧に追加列として表示

3. **パフォーマンス悪化区間のハイライト**
   - 討伐平均秒数が悪い区間を色分け表示
   - 閾値設定UI（下位n%、下位n位、絶対値の3パターン）

4. **精鋭数カウンター**
   - 「現在の合計/全区間の合計」をリアルタイム表示
   - 完了した区間の精鋭数を自動集計

#### データ管理
- 既存のLiveSplit保存機能を拡張
- スプリット情報に精鋭数フィールドを追加
- 後方互換性を維持

詳細は [FEATURES.md](./FEATURES.md) および [CUSTOMIZATION.md](./CUSTOMIZATION.md) を参照。
